# 🚨 HearMe

HearMe is a simple emergency web application built using HTML, CSS, and JavaScript.

It helps users quickly send voice alerts and share their live location using browser APIs.

---

## 📌 Features

- Voice alerts for emergency messages (Help, Ambulance, I can’t speak)
- Text-to-speech using browser
- Live location sharing using Geolocation API
- Simple and fast one-click interface

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Speech Synthesis API
- Geolocation API

---

## 🎯 Purpose

This project is designed to improve emergency communication speed using simple web technologies. It allows users to send quick voice alerts and share their location when they are unable to speak or react normally.

---

## 🚀 How to Use

- Open the application in a browser
- Click any emergency button
- The message will be spoken aloud
- Click "Share Location" to get your current coordinates
